# blackline

`blackline` is a sleek command-line system resource monitor with aesthetic visual bars and comprehensive stats.

## Features
- **Visual Bars** — Color-coded progress bars (green/yellow/red) for memory, disk, and CPU
- **Top Processes** — Show top N memory-consuming processes with `--top 5`
- **Delta Mode** — Track changes between intervals with `--delta`
- **Per-Core CPU** — Individual core usage statistics
- **Multiple Styles** — Choose between `bars`, `compact`, or `minimal` output
- **JSON Output** — Machine-readable format for scripting with `--json`
- **Human-Readable** — Easy-to-parse units (MiB/GiB) with `--human`
- **Timestamp** — Add timestamps to each update with `--timestamp`
- **Continuous Monitoring** — Real-time updates with configurable intervals

## Installation
```bash
cargo install --path .
```

## Usage
```bash
# One-time report with visual bars
blackline

# Continuous monitoring every second (default)
blackline --interval 1

# Show top 5 memory-consuming processes
blackline --top 5

# Track changes between intervals
blackline --delta --interval 2

# Show timestamp for each update
blackline --timestamp

# Minimal output (just percentage)
blackline --style minimal

# JSON output for scripting
blackline --json

# Show all stats with CPU
blackline --all --cpu

# Custom bar count
blackline --bars 30
```

## CPU Stats
The CPU usage is calculated using all available fields from `/proc/stat`:
- user, nice, system, idle, iowait, irq, softirq, steal

This provides accurate CPU usage even under I/O or VM load.

## Cross-Platform Support
- **Linux**: Reads `/proc/meminfo`, `/proc/stat`, and `/proc/net/dev`.
- **macOS**: Uses `sysctl`, `vm_stat`, `top`, `pmset`, and `netstat`.
- Other platforms: Currently unsupported.
