# blackline

`blackline` is a tiny command‑line utility that reports memory usage on the host system.

## Features
- Shows total, available, and used memory.
- Human‑readable (`--human`) or JSON (`--json`) output.
- Continuous monitoring with `--interval <seconds>`.
- Quiet mode (`--quiet`) for script‑friendly usage.
- Cross‑platform detection – runs on Linux and reports a clear error on other OSes.

## Installation
```bash
cargo install --path .
```

## Usage
```bash
# One‑time report in human units
blackline --human

# JSON output (useful for tooling)
blackline --json

# Continuous report every 5 seconds
blackline --interval 5

# Suppress normal output (only errors are printed)
blackline --quiet --human
```

## Cross‑platform support
- On Linux the tool reads `/proc/meminfo`.
- On non‑Linux platforms it exits with a clear message: "Unsupported platform: blackline only supports Linux".
